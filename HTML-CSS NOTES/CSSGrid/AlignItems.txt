<---------------------------------Align Items---------------------------------->

Property:
align-items - align all items vertically at once

Values:
(default) stretch - stretches content to full height of the cell
start - aligns the content at the top of the cell
center - aligns the content at the center of the cell
end - aligns the content at the end of the cell

Example: align-items: center;

//----------------------------------EXAMPLE-----------------------------------//

<style>
  .item1{background:LightSkyBlue;}
  .item2{background:LightSalmon;}
  .item3{background:PaleTurquoise;}
  .item4{background:LightPink;}
  .item5{background:PaleGreen;}

  .container {
    font-size: 40px;
    min-height: 300px;
    width: 100%;
    background: LightGray;
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-rows: 1fr 1fr 1fr;
    grid-gap: 10px;
    /* Only change code below this line */
    align-items: end;  //NOTE
    /* Only change code above this line */
  }
</style>
